ngx-material-dashboard

Components built on top of Angular Material, annotations and services for interfacing with JSON APIs, and a testing library to simplify writing tests for those components and services

widgets

Components designed on top of Angular Material to provide additional functionality, including basic CRUD capabilities, with minimal code while adhering to Material design principles.

JSON

Libraries designed to make interfacing with JSON APIs easy, and allow you to work with classes and services instead of HttpClient and JSON directly.

testing

Testing library designed to simplify writing tests for the widgets library using Page element objects that eliminate the need for a lot of boilerplate set up so you can focus on the tests themselves.
