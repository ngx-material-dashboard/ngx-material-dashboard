ngx-material-dashboard

Annotations and services for interfacing with JSON APIs, Components built on top of Angular Material, and a testing library to simplify writing tests

JSON

Libraries designed to make interfacing with JSON APIs easy, and allow you to work with classes and services instead of HttpClient and JSON directly.

widgets

Components designed on top of Angular Material to provide additional functionality, including basic CRUD capabilities, with minimal code.

testing

Testing library designed to simplify writing tests for the widgets library eliminating a lot of boilerplate so you can focus on the tests themselves.
